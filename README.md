## Quick Start

### Install as Claude Code Plugin

```bash
# Install from marketplace
claude plugins install everything-claude-marketing

# Or clone and install locally
git clone https://github.com/YOUR_USERNAME/everything-claude-marketing.git
cd everything-claude-marketing
claude plugins install .
```

### Try It

```bash
# Plan a campaign
claude> /campaign launch our new AI feature to mid-market SaaS companies

# Write copy
claude> /copy hero section for our landing page — target audience is startup founders

# Run SEO audit
claude> /seo-audit analyze our blog post about email marketing

# Design email sequence
claude> /email-sequence 5-email welcome series for new trial signups

# Create social content
claude> /social-post announce our Series A on LinkedIn and X
```

## Core Principles

1. **Strategy-First** — Start with objectives and audience before tactics
2. **Data-Driven** — Back claims with metrics, benchmarks, and evidence
3. **Brand-Consistent** — Every output aligns with brand voice and guidelines
4. **Compliance-Aware** — GDPR, CAN-SPAM, FTC, DSGVO always respected
5. **Platform-Native** — Content adapts per platform, never cross-posted verbatim
6. **Ethical Marketing** — No misleading claims, dark patterns, or manipulative tactics

## Multi-Agent Workflows

Complex tasks automatically orchestrate multiple agents:

| Workflow | Agents Involved |
|----------|----------------|
| **Campaign Planning** | strategist → content-strategist + copywriter + paid-ads-optimizer |
| **Product Launch** | product-marketer → copywriter + pr-specialist + social-media-manager + email-marketer |
| **Rebranding** | brand-guardian + creative-director → content-strategist + social-media-manager |
| **Growth Sprint** | growth-hacker → analytics-interpreter + cro-specialist + paid-ads-optimizer |
| **Content Program** | content-strategist → copywriter + seo-analyst + social-media-manager |

## Contexts

Switch Claude's behavior based on your current mode:

| Context | Mode | Behavior |
|---------|------|----------|
| `campaign` | Active campaign work | Write first, optimize after. Focus on output volume and speed. |
| `research` | Market research & analysis | Read widely, document findings, don't jump to tactics. |
| `review` | Content & brand review | Evaluate quality, flag issues by severity, suggest fixes. |
