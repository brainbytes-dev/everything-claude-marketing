# /copy

Write high-converting marketing copy for any format, from landing page headlines to long-form sales pages. Adapts voice, tone, and structure to your brand and audience.

## What This Command Does

The `/copy` command generates polished marketing copy tailored to your specified format, audience, and goals. It handles everything from short-form ad copy to full landing pages, applying proven copywriting frameworks and persuasion principles. The output is ready to use or serves as a strong first draft for your team to refine.

The command delegates to the **copywriter agent**, which draws on direct-response and brand copywriting techniques to produce compelling, goal-oriented content.

## When to Use

- You need landing page copy — hero sections, feature blocks, CTAs, or full pages
- You are writing ad copy for any platform (Google, Meta, LinkedIn, etc.)
- You need email copy — promotional, transactional, or newsletter content
- You want sales page copy with structured persuasion flow
- You need social media captions or post copy
- You are drafting website copy for any section (about, pricing, features, etc.)
- You want to rewrite existing copy for better performance

## How It Works

1. **Context Gathering** — Identifies the format, audience, product/service, and desired action
2. **Framework Selection** — Applies the appropriate copywriting framework (AIDA, PAS, BAB, StoryBrand, or others based on format)
3. **Voice Calibration** — Matches your brand voice and tone guidelines, or establishes an appropriate tone if none are provided
4. **Draft Creation** — Writes the full copy with headlines, body, proof elements, and calls to action
5. **Variant Generation** — Provides alternative headlines or CTA options where applicable
6. **Optimization Notes** — Includes suggestions for testing and improving performance

## Important Notes

- Always specify the format (landing page, email, ad, etc.) and the target audience — generic requests produce generic copy
- Include your unique value proposition, key differentiators, or competitive advantages for stronger output
- If you have brand voice guidelines, share them — the copywriter agent will match tone, vocabulary, and style
- The command produces copy, not design — pair with a designer for layout and visual hierarchy decisions
- For ad copy specifically, consider using `/ad-creative` which combines copy with targeting and creative direction
- Generated copy should always be reviewed for factual accuracy, especially claims, statistics, and product capabilities
