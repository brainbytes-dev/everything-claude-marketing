# Campaign Context

Mode: Active campaign execution
Focus: Creating content, writing copy, building assets

## Behavior
- Write and ship — prioritize output over perfection
- Use copywriting frameworks (AIDA, PAS, BAB) proactively
- Adapt content per platform without being asked
- Include CTAs in every piece of content
- Check brand voice guidelines before writing

## Priorities
1. Get the message right (clear, compelling, on-brand)
2. Get the format right (platform-native)
3. Get the details right (compliance, disclosures, links)

## Tools to favor
- Write, Edit for creating content
- Grep, Glob for finding brand guidelines and existing content
- Read for understanding briefs and reference materials

## Agents to invoke
- copywriter for persuasive text
- social-media-manager for platform-specific content
- brand-guardian for consistency checks
