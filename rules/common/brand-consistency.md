# Brand Consistency

## Voice & Tone

ALWAYS maintain brand voice across all outputs:

- **Voice is constant** — the brand's personality doesn't change (e.g., "confident and approachable")
- **Tone adapts** — the emotional register shifts by context (e.g., celebratory for wins, empathetic for support)
- **Vocabulary matters** — use the brand's preferred terms consistently (e.g., "customers" vs "users" vs "members")
- **Avoid generic marketing speak** — no "synergy," "leverage," "best-in-class" unless it's genuinely the brand's voice

## Voice Consistency Rules

1. **Check brand guidelines first** — before writing any copy, reference existing brand voice documentation
2. **Match existing content** — new content should feel like it comes from the same source as existing content
3. **Maintain reading level** — if the brand writes at a 6th-grade level, don't suddenly produce academic prose
4. **Consistent formatting** — if the brand uses sentence case for headlines, never use title case
5. **Emoji and punctuation style** — follow the brand's established patterns (exclamation points, emoji usage, Oxford comma)

## Brand Architecture

- **Master brand**: The primary brand identity governs all sub-brands
- **Sub-brands**: Must follow master brand guidelines with approved variations
- **Product names**: Use exact capitalization and formatting (e.g., "iPhone" not "IPhone")
- **Taglines**: Use verbatim — never paraphrase or modify taglines

## Visual Consistency

When creating content briefs or creative direction:
- Reference approved color palette (primary, secondary, accent colors)
- Use approved typography (headline font, body font, weights)
- Follow approved imagery style (photography style, illustration style)
- Maintain consistent logo usage (clear space, minimum size, color variants)
- Apply approved layout patterns and grid systems

## Messaging Hierarchy

All content must align with the approved messaging hierarchy:
1. **Brand promise** — the core value proposition
2. **Messaging pillars** — 3-5 key themes that support the promise
3. **Proof points** — evidence and examples for each pillar
4. **Objection handling** — pre-approved responses to common concerns

## Quality Gate

Before any content is finalized:
- [ ] Voice matches brand guidelines
- [ ] Tone is appropriate for the context and audience
- [ ] Brand-specific terminology used correctly
- [ ] Visual direction aligns with brand standards
- [ ] Messaging supports approved hierarchy
- [ ] No off-brand language or claims
- [ ] Formatting follows brand conventions
