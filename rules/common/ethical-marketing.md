# Ethical Marketing

## Core Principle

Marketing should help people make informed decisions. It should never manipulate, deceive, or exploit.

## Prohibited Practices

### Misleading Claims
- NEVER make claims that cannot be substantiated with evidence
- NEVER use fake statistics or fabricated social proof
- NEVER imply endorsements that don't exist
- NEVER use misleading before/after comparisons
- NEVER exaggerate product capabilities or results

### Dark Patterns
- NEVER use confirmshaming ("No thanks, I don't want to save money")
- NEVER use hidden costs revealed only at checkout
- NEVER make unsubscribe intentionally difficult
- NEVER use forced continuity (hard-to-cancel subscriptions)
- NEVER use trick questions or double negatives in opt-ins
- NEVER disguise ads as editorial content without disclosure
- NEVER use bait-and-switch tactics

### Artificial Urgency & Scarcity
- NEVER create fake countdown timers that reset
- NEVER fabricate limited availability ("Only 2 left!" when there are 2000)
- NEVER use fake "someone just bought" notifications
- Real urgency is fine — state the genuine reason (seasonal sale, limited production run)

### Targeting & Privacy
- NEVER target vulnerable populations with exploitative messaging
- NEVER use personal data in ways the user hasn't consented to
- NEVER retarget with sensitive health, financial, or personal data
- NEVER market adult products or services to minors

## Required Practices

### Transparency
- Clearly disclose paid partnerships, affiliates, and sponsored content
- Show real pricing — no hidden fees
- Use genuine testimonials from real customers
- Disclose material connections between endorsers and brands

### Honesty
- Make only claims you can substantiate
- Show representative results, not only best-case outcomes
- If results vary, say so clearly
- Correct errors publicly when mistakes are made

### Respect
- Honor unsubscribe and opt-out requests immediately
- Respect "Do Not Track" and privacy preferences
- Give users control over their data and preferences
- Avoid excessive frequency — respect people's attention

### Inclusivity
- Use inclusive language and imagery
- Ensure accessibility in all marketing materials
- Represent diverse audiences authentically
- Avoid stereotypes and tokenism

## Decision Framework

When in doubt about an ethical gray area, ask:
1. **Would I be comfortable if this practice were made public?**
2. **Would I want to be on the receiving end of this tactic?**
3. **Does this help the customer make a better decision, or does it manipulate them?**
4. **Would this damage trust if the customer found out how it works?**

If any answer raises concern, don't do it.

## Quality Gate

Before publishing:
- [ ] All claims are substantiated
- [ ] No dark patterns present
- [ ] Urgency/scarcity is genuine
- [ ] Disclosures are clear and conspicuous
- [ ] Targeting is ethical and appropriate
- [ ] Content is inclusive and accessible
- [ ] Pricing is transparent
